关于css左右布局，大家应该都很有体会。左边一个图，右边文字，类似下图的效果：
实现方法也很多，我平时使用的方法主要有两个：

  • 左右浮动.

  • 可以左边绝对定位，然后把右边的margin-left,或者padding-left设置一下。

    这两个方法有利有弊。浮动兼容性最好，但是会对后面的元素产生影响，要清除浮动。绝对定位，书写比较复杂。
